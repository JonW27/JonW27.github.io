HTML Overview

HTML stands for Hypertext Markup Language. It is used in building website ands doing web design. As tech expanded and became more advanced HTML had to advance also. This led to different variations of HTML with different !DOCTYPE declarations. The one I will be teaching you is the latest one, HTML5.

Format and General Rules

All HTML documents need at minimum the following things:

  • !DOCTYPE
  • <html> and its closing tag
  • <head> and its closing tag
  • <title> and its closing tag
  • <body> and its closing tag

The bare skin is shown below:

<DOCTYPE html>
<html>
  <head>
    <title>Your Title</title>
  </head>  
  <body>
  </body>
</html>

Content

As shown in the SVG above (I'll explain a SVG later) you should have gotten nothing. Why is that? You simply have no content. As should be expected there are content elements and container elements for them. A list is below:

  • The paragraph element <p>
  • THe heading elements <h1> <h2> <h3> <h4> <h5> <h6>
  • The link element <a>
  • The image element <img>
  • The table element <table>
  • The form element <form>
  • The ordered list element <ol>
  • The unordered list element <ul>

Each of these elements (these are not the only ones) have attributes to help determine things. For example, the <a> element has the attribute href which tells what url the link will link to and target which tells if the url will be opened in a new tab. Elements like <table> have elements like <tr> which defines a table row and <td> which defines a single cell. The elements <ol> and <ul> also have the subelement <li> which defines a single list item. All of these elements, attributes, and subelements will be discussed and placed in a to-be-made Reference page.
