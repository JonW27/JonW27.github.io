HTML Overview

HTML stands for Hypertext Markup Language. It is used in building website ands doing web design. As tech expanded and became more advanced HTML had to advance also. This led to different variations of HTML with different !DOCTYPE declarations. The one I will be teaching you is the latest one, HTML5.
