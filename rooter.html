Rooter documentation

Rooter is a simple Applescript tool for Mac created by Jonathan Wong. The download link is available here: Rooter. Please read the warranty near the bottom of this page first. This is the following documentation page



Installation

Once you visit the link click the "this file has been truncated, but you can view the full file..." link. It will download the dmg. Following that click the dmg to open it and then drag the application into your applications folder. Voila! You're done.
